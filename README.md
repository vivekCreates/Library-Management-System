# Library Management System

A web-based application for managing library resources, users, and transactions.

## Features

- User authentication and role-based access
- Book catalog management (add, update, delete, search)
- Book issue and return tracking
- Fine calculation for overdue books

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/vivekCreates/Library-Management-System.git
    cd Library-Management-System
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Configure environment variables:**
    - Copy `.env.local` to `.env` and update values as needed.

4. **Run the application:**
    ```bash
    npm run dev
    ```
